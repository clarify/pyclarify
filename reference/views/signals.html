class Signal(*, name: str, description: str = '', labels: Dict[str, List[str]] = {}, sourceType: SourceTypeSignal = SourceTypeSignal.measurement, valueType: TypeSignal = TypeSignal.numeric, engUnit: str = '', enumValues: Dict[str, str] = {}, sampleInterval: timedelta | None = None, gapDetection: timedelta | None = None, annotations: Dict[str, str] | None = None)[source]ÔÉÅ

Model for sending Signal metadata to Clarify.

Parameters:

  • name (string(len:1-128)) ‚Äì A human-readable name for the resource.

  • description (string(len:0-1000)) ‚Äì A free-form description of the resource.

  • labels (Labels) ‚Äì A map of custom classification attributes. Filtering is done on label keys (labels.<key>).

  • sourceType (string(enum)) ‚Äì Classification of the data source. The value must be ‚Äúaggregate‚Äù, ‚Äúmeasurement‚Äù or ‚Äúprediction‚Äù.

  • valueType (string(enum)) ‚Äì How to interpret time-series data points. The value must be ‚Äúenum‚Äù or ‚Äúnumeric‚Äù.

  • engUnit (string) ‚Äì Engineering unit for time-series data in numeric representations.

  • enumValues (map(string => string(len:1-128))) ‚Äì Map of numeric values to display text in enum representations. The key must represent an integer in range 0-9999.

  • sampleInterval (Fixed Duration, null) ‚Äì The expected distance between data points.

  • gapDetection (Fixed Duration, null) ‚Äì The maximum distance between two data-points before considering it to be a gap.

  • annotations (Annotations) ‚Äì A key-value store where integrations can store programmatic meta-data about the resource instance. Filtering is done on member fields.

Example

>>> from pyclarify import Signal

Creating a signal a minimal signal.

>>> signal = Signal(name="My new signal")

Creating a signal with all attributes set.

>>> signal = Signal(
...     name="My new signal"
...     description="This signal is an example."
...     labels={
...         "environment": ["dev", "mocking"],
...         "unit":["cloud"]
...     }
...     engUnit="‚ÑÉ"
...     sampleInterval="PT30S"
...     gapDetection="PT5M"
