What is PyClarify?Â¶

PyClarify is a tool that lets you easily interact with Clarify.

You can:

  • Insert data to signals / items

  • Insert metadata to sigals / items

  • Read data and metadata from items

  • Read data and metadata from signals

  • Publishing signals
